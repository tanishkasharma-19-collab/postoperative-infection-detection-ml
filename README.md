# Postoperative Infection Detection using Machine Learning

## Overview

This project focuses on early prediction of postoperative infections using machine learning techniques on clinical perioperative data.

## Problem Statement

Postoperative infections increase hospital stay, cost, and mortality risk. Early detection can help clinicians take preventive actions.

## Dataset

Clinical dataset containing:

* Patient demographics
* Surgical variables
* Comorbidities
* Intraoperative temperature readings

## Feature Engineering

* Temperature drop metrics
* Average intraoperative temperature

## Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* XGBoost
* Stacking Ensemble
* Weighted Voting Ensemble

## Techniques Used

* Data preprocessing
* One-hot encoding
* Standard scaling
* SMOTE for imbalance handling
* Threshold tuning for recall optimization

## Results

Logistic Regression achieved highest recall for infection detection.
